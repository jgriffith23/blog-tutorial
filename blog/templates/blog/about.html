About

Hi! My name is Jennifer, and I am a software engineer with a background in electrical engineering. I spent several years editing technical books, and for a while, I was a freelance technical editor and writer. After that, I spent well over a year teaching full-stack web development in Python and JavaScript as a lab instructor at Hackbright Academy.

I love Python, and I'm still building apps in it today. Beyond maintaning my Python skills, my engineering interests include back-end web development, infosec, databases, and infrastructure. One of these days I'll re-learn Java and C, and I'm trying to decide on a functional programming language to pick up.

I'm currently seeking my next adventure in software engineering! If you're hiring, don't hesitate to drop a line on LinkedIn.

I started my original GitHub blog to document my experiences as a student at Hackbright. Post-graduation, I built this Django site to try out a new framework. I'm moving the old material over (still a work in progress) and will continue using the blog to:

  • Keep my writing skills (both technical and non-technical) fresh
